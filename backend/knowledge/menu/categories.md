# 菜单分类

本店菜单分为以下七大类，方便您根据口味和用餐场景快速选择。

| 分类 | 排序 | 说明 |
|------|------|------|
| 热菜 | 1 | 川味热炒，锅气十足，麻辣鲜香 |
| 凉菜 | 2 | 开胃小菜，清爽解腻，佐餐必备 |
| 素菜 | 3 | 清淡健康，原汁原味，老少皆宜 |
| 海鲜 | 4 | 鲜活海产，清淡鲜美，每日直采 |
| 汤品 | 5 | 暖心汤羹，滋补养生，暖胃解腻 |
| 主食 | 6 | 饱腹主食，经典口味，甜咸兼具 |
| 饮品 | 7 | 解辣饮品，清爽甘甜，餐餐相配 |

## 点餐建议

- **喜欢麻辣重口**：优先浏览热菜，如招牌水煮鱼、毛血旺、辣子鸡。
- **不吃辣**：可选择素菜、海鲜、汤品、主食中的不辣菜品。
- **家庭聚餐**：推荐搭配一份大菜（如东坡肘子、干烧大黄鱼）、两道热菜、一道素菜和一份汤。
- **工作简餐**：一荤一素一主食即可，如宫保鸡丁 + 白灼菜心 + 白米饭。
